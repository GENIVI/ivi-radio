Content

This Tuner API simulation consists of the following parts:

bin : Directory, where executables build will be placed in; additionally contains 'config' subdirectory with station data
doc : this directory contains a readme, a testplan, header files and images for the doxygen documentation land the doxygen documentation itself if built
src-gen : automatically generated code; created by export from Enterprise Architect, import in Yamaica tool, tweaking and generation of C++ standard and DBUS specific parts
src : handcrafted sourcecode files representing Tuner Middleware and HMI include : handcrafted header files representing Tuner Middleware and HMI
Makefile.am
configure.ac
create_doxygen_docu.sh
etc. : Build configuration templates for autoconf/automake

Build and run

  1. Install Ubuntu Linux (g++ 4.7.2 required)
  2. Install CommonAPI as per instructions (tested with 2.0.3) Do not forget to create patched DBUS packages, install them and reboot machine once afterwards
  3. Install this software in $DIR (choose directory of your choice)
    cd $DIR
    autoreconf -i
    ./configure
    make
  4. Run tests
    # in one console/xterm window
    cd $DIR/bin
    ./Middleware
    # in another console/xterm window
    cd $DIR/bin
    ./Client 1 # start HMI client simulation (will not return)
    # if needed, in yet another console/xterm window
    cd $DIR/bin
    ./Client 2 # start second HMI client simulation (will not return)
    # in one console/xterm window
    cd $DIR/bin
    ./Middleware
    # in another console/xterm window
    cd $DIR/bin
    ./Client 2 # start second HMI client simulation (will not return)
  5. alternatively run testcases with python script automatically. This will run a set of testcases described in the python script (e.g. Seek, GetList, configure announcement,...) to clear all preconditions to continue manual testing. After executing the Testcases the automatic part ends and the Client waits for manual console input.
    # in one console/xterm window
    cd $DIR/bin
    ./Middleware
    # in another console/xterm window
    cd $DIR/bin
    ./autotest.py
